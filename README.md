# 🌍 LABELA

## Sistem Laporan Bencana Alam

LABELA adalah aplikasi berbasis web yang dibuat untuk membantu masyarakat dalam melaporkan dan mengelola informasi bencana alam secara cepat dan efisien.

Aplikasi ini dikembangkan menggunakan Laravel, Tailwind CSS, dan MySQL.

---

# ✨ Fitur Utama

- ✅ CRUD Laporan Bencana
- ✅ Dashboard Statistik
- ✅ Search Laporan
- ✅ Detail Laporan
- ✅ Status Penanganan
- ✅ Tampilan Modern UI
- ✅ Responsive Design
- ✅ Navbar & Footer Modern

---

# 📸 Tampilan Aplikasi

## Dashboard
Menampilkan statistik total laporan, proses penanganan, dan laporan selesai.

## Daftar Laporan
Menampilkan seluruh data laporan bencana.

## Detail Laporan
Menampilkan informasi lengkap mengenai laporan bencana.

## Profil & Kontak
Berisi informasi mengenai aplikasi LABELA.
